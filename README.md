
# Parking System

Projeto acadêmico em desenvolvimento para a disciplina de Engenharia de Software.

---

##  Sobre o Projeto

O **Parking System** é um sistema de estacionamento automatizado desenvolvido para controlar a entrada e saída de veículos, gerenciar vagas disponíveis e organizar o fluxo do estacionamento de forma eficiente.

---

##  Objetivos

- Melhorar o controle do estacionamento
- Reduzir erros no registro de veículos
- Automatizar o gerenciamento de vagas
- Agilizar o atendimento aos usuários

---

##  Funcionalidades

- Verificação de vagas disponíveis
- Registro de entrada de veículos
- Identificação automática de placas
- Controle de saída de veículos
- Gerenciamento de vagas
- Emissão de ticket
- Controle de lotação
